
# Decentralized AI test

A task to impliment parallax effects.


## Run Locally

Clone the project

```bash
  git clone https://github.com/GrigoryyKach/decentralized-ai.git
```

Go to the project directory

```bash
  cd decentralized-ai
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
